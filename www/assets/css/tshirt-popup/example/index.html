T-Shirt Pop Up

By default, T-Shirt Pop Up rely on Font Awesome and Animate.css to add graphics and animation into the popup. It use minimal styling to easy overide the CSS. T-Shirt Pop Up design to load content for external link and append it into the document.

Basic usages

  1. Basic popup. Just add the function to the jquery selector, it will take care everything. 

    $("a").popup();

  2. Popup with width and height. By default, the popup will use the content width and height, you can either specific the width or height in the CSS or set the paramater like this: 

    $("a").popup({
	width: 400,
	height: 100
});

  3. Popup with different show animation. For list of animation that you can use, please visit animate.css.

    $("a").popup({
	animation: "bounceInUp"
});

  4. Basic popup with custom class. You can set the content div class name into something that you We style the content with orange color.

    $("a").popup({
	className: "customClass"
});

  5. Loading something that unvailable or url from different domain will produce a broken link window.

You can view the source code for more information about popup features.
